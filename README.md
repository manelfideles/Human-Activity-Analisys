# Human-Activity-Analysis

No âmbito da cadeira de Análise e Transformação de Dados, foi-nos proposto
analisar uma gama de 12 atividades através de sinais detetados pelo giroscópio e
acelerómetros de um telemóvel colocado na cintura. Estas atividades podem-se dividir em 3
grupos distintos, dinâmicos (Walking (W), Subindo Escadas (WU), Descendo Escadas
(WD)), estáticos (Sentado (SIT), De pé (STAND), Deitado (LAY)) e por fim de transição (De
pé - Sentado (STAND2SIT), Sentado - De pé (SIT2STAND), Sentado - Deitado (SIT2LIE),
Deitado - Sentado (LIE2SIT), De pé - Deitado (STAND2LIE), Deitado - De pé
(LIE2STAND)).
Estas atividades foram todas captadas com um Dataset SBHAR com 2
acelerómetros e 1 giroscópio num smartphone Samsung S2, recolhendo dados dos 3 eixos
espaciais com um frequência de amostragem, fs, de 50 Hz.

Todos os gráficos aqui exibidos são referentes à experiência 1 (à exceção das
tarefas onde são precisos dados de todas as tarefas).
